分页查看文本内容命令：less

less命令 的作用与more十分相似，都可以用来浏览文字档案的内容，不同的是less命令允许用户向前或向后浏览文件，而more命令只能向前浏览。用less命令显示文件时，用PageUp键向上翻页，用PageDown键向下翻页。要退出less程序，应按Q键。

语法

less(选项)(参数)

参数

  • 文件：指定要分屏显示内容的文件。

选项

无

j：下滚一行

k：上滚一行

b: 回翻一页

f: 下翻一页

q: 退出less 命令

/ :搜索

:n 浏览下一个文件

:p 浏览上一个文件

history | less ：查看最近使用的命令






查看文件开头内容命令：head

head命令 显示文件的开头部分。

语法

head [OPTION]… [FILE]…

参数

  • FILE（可选）：要处理的文件，可以为一或多个。

选项

显示文件前10行内容（默认）

-n 5 显示文件前5行内容

-c 20 显示前20个字符






查看文本尾部内容命令：tail

tail命令 在屏幕上显示指定文件的末尾若干行

语法

tail (选项) (参数)

参数

  • 文件列表：指定要显示尾部内容的文件列表。

选项

默认10行

-n 5 显示文件尾部5行内容

-c 20 显示文件尾部20字符内容

-f 动态显示文件的最后10行内容

CTRL+C结束






反向显示文本内容命令：tac

tac命令 连接多个文件并以行为单位反向打印到标准输出。

语法

tac [OPTION]… [FILE]…

参数

  • FILE（可选）：要处理的文件，可以为一或多个。

选项

无 反向输出，与cat相反






统计文件行号命令：nl

nl命令 为每一个文件添加行号。

语法

nl [OPTION]… [FILE]…

参数

  • FILE（可选）：要处理的文件，可以为一或多个。

选项

无 加行号，空行不加

-b a 加行号 空行也加

-b a -n rz 行号在自己栏位的最右侧显示，且加0对齐格式

-w 设置行号宽度






统计文本字数信息命令：wc

wc命令 统计指定文件中的字节数、字数、行数，并将统计结果显示输出。利用wc指令我们可以计算文件的Byte数、字数或是列数，若不指定文件名称，或是所给予的文件名为“-”，则wc指令会从标准输入设备读取数据。wc同时也给出所指定文件的总统计数。

语法

wc(选项)(参数)

wc [选项]… [文件]…

wc [选项]… –files0-from=F

参数

  • 文件：需要统计的文件列表。

选项

无 统计文件的行数，字数以及字节数

-w 只统计字数

-m 只统计字符数

-c 只统计字节数

-l 只统计行数

-L 打印文件最长行的长度 （不包含不可见的字符）

使用管道统计文本行数






文件切割命令：split

split命令 可以将一个大文件分割成很多个小文件，有时需要将文件分割成更小的片段，比如为提高可读性，生成日志等。

语法

split [option] … [file[prefix]]

参数

  • 文件

-number 将文件每几行切割成一个小文件

-b 按大小切割

-d -a 3 以数字作为后缀，并指定后缀宽度为3

为分割后的文件指定文件名的前缀






文本截取命令：cut

cut 命令 用来显示行中的指定部分，删除文件中指定字段。cut 经常用来显示文件的内容，类似于 type 命令。

语法

cut（选项）（参数）

参数

  • 文件：指定要进行内容过滤的文件。

选项

-f 提取第几列的内容

–complement 提取除了第二行的其他内容

-d ‘;’ 使用-d指定字符分隔符

提取2,3,4,6个字节

-cnumber1-number2提取一段字符

-c-number 提取前number个字符

-cnumber- 提取第number个之后的字符

-c和-b遇到中文字符可能会乱码






文件合并命令：paste

paste命令 用于将多个文件按照列队列进行合并。

语法

paste(选项)(参数)

参数

  • 文件列表：指定需要合并的文件列表。

选项

无 默认tab

-d ‘:’ 以:为分隔符

-s 以行分割

ls | paste -d ‘ ‘ - - - 以三列分割






文本内容排序命令：sort

sort命令 对文本文件中所有行进行排序

语法

sort [OPTION]… [FILE]…

sort [OPTION]… –files0-from=F

参数

  • FILE（可选）：要处理的文件，可以为任意数量。

选项

无 按第一个字母排序 数字也是

-n 按数字从小到大排序

-r 反向排序

-t ‘;’ -k 3 -n 以;为分隔符，对指定的文件内容按照数字大小对第三列进行排序






去除重复行命令：uniq

uniq命令 显示或忽略重复的行。

语法

uniq [OPTION]… [INPUT [OUTPUT]]

参数

  • INPUT（可选）：输入文件，不提供时为标准输入。
  • OUTPUT（可选）：输出文件，不提供时为标准输出。

选项

无

-c 打印重复次数

-d 只记录有重复的

-u 只记录没重复的






比较差异_打补丁命令：diff_patch

diff命令 在最简单的情况下，比较给定的两个文件的不同。如果使用“-”代替“文件”参数，则要比较的内容将来自标准输入。diff命令是以逐行的方式，比较文本文件的异同处。如果该命令指定进行目录的比较，则将会比较该目录中具有相同文件名的文件，而不会对其子目录文件进行任何比较操作。

语法

diff(选项)(参数)

参数

  • 文件1：指定要比较的第一个文件；
  • 文件2：指定要比较的第二个文件。

选项

无

第一个文件第三行和第二个文件第三行有差异

第一个文件第五行和第二个文件第六行有差异


-y -W 50 并排显示

‘|’ 为区别

‘>’ ‘<’ 表示哪边多出来的

-c 显示全部内文，并标出不同之处

生成补丁

打补丁

打完补丁后f1.txt 和f2.txt一样






连接两个文件命令：join

join命令 用来将两个文件中，制定栏位内容相同的行连接起来。找出两个文件中，指定栏位内容相同的行，并加以合并，再输出到标准输出设备。

语法

join(选项)(参数)

参数

  • 文件1：要进行合并操作的第1个文件参数；
  • 文件2：要进行合并操作的第2个文件参数。

选项

无

以最前面的数字为标志合并相同数字行的内容

-a1 -a2 加上不同的标志行内容


-o 1.2 1.3 2.3 选择第一个文件第二列、第三列，第二个文件第三列合并

-t ‘ ‘ 以空格为分隔符

-v1 显示前一个文件不相同标志行

-v2 显示后一个文件不相同标志行

-j number 以第几列为标志连接






复制命令：cp

cp命令 用来将一个或多个源文件或者目录复制到指定的目的文件或目录。它可以将单个源文件复制成一个指定文件名的具体的文件或一个已经存在的目录下。cp命令还支持同时复制多个文件，当一次复制多个文件时，目标文件参数必须是一个已经存在的目录，否则将出现错误。

语法

cp(选项)(参数)s

参数

  • 源文件：制定源文件列表。默认情况下，cp命令不能复制目录，如果要复制目录，则必须使用-R选项；
  • 目标文件：指定目标文件。当“源文件”为多个文件时，要求“目标文件”为指定的目录。

选项

无

-r 文件夹复制

-i 提示

-b 备份原来的尾部加~

-a 将源文件属性全部复制






显示目录或文件大小命令：du

du命令 也是查看使用空间的，但是与df命令不同的是Linux du命令是对文件和目录磁盘使用的空间的查看，还是和df命令有一些区别的。

语法

du (选项)(文件)

选项

无

-h 显示具体字节

-ah 查看所有文件夹下所有文件

-s 仅显示文件夹的大小

–max-depth=0 深度为0（即只看本文件）
